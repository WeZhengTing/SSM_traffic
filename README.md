# SSM_traffic
# 一、介绍
> 一个练手的SSM框架项目，使用idea搭建的

# 二、实现功能
> 1.实现用户管理员的登录、注册、退出
> 2.使用数据库完成的分页
> 3.实现用户增删改查,还有条件查询
> 4.实现罚单的处理流程（还未完善,只是搭了个架子）
